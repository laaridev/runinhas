# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Runinhas, please report it by emailing the maintainer or opening a private security advisory on GitHub.

**Please do not open public issues for security vulnerabilities.**

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Security Features

- **CodeQL** - Automated security scanning on every commit
- **Semgrep** - SAST analysis for Go and TypeScript
- **Dependency Review** - Automatic vulnerability detection in dependencies
- **No telemetry** - Zero data collection, completely local application

## Best Practices

- Keep your ElevenLabs API key secure
- Download releases only from official GitHub releases
- Verify checksums of downloaded binaries
